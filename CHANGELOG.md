## [0.1.0-skeleton] - 2026-04-30

### Ajouté
- Squelette initial du projet Android (Kotlin + Camera2)
- Documentation stratégique : README, ROADMAP, LIMITATIONS, CONTRIBUTING
- Architecture logicielle documentée dans `docs/architecture/`
- Bibliographie scientifique de 19 références dans `docs/research/`
- Protocole méthodologique de recherche LLM dans `docs/protocols/`
- Rapport de vérification de la bibliographie
- Planches A4 de marqueurs ArUco prêtes à imprimer
- Configurations Gradle et Android Manifest
- Workflows GitHub Actions de base (lint, build, test)
- Licences GPL-3.0 (code) et CC-BY-4.0 (documentation)

### Décidé
- Stack technique : Kotlin natif Android avec Camera2 API
- Hardware cible v0.1 : Samsung Galaxy Note 20 Ultra
- Hardware cible v0.2 : Ulefone Armor 28 Ultra Thermal
- Précision communicationnelle : multi-niveaux selon le type de cote
- Pas de cloud propriétaire, pas de télémétrie

### Connaissance
- Validation scientifique des 7 pistes techniques retenues
- Identification des pôles académiques pertinents (Suisse, Chine, Allemagne, Italie)
- Validation empirique du protocole de vérification (12/19 références confirmées 100%)
